## EO Dashboard Presentation Materials
Welcome to the central repository for eodashboard.org/race.esa.int presentation materials. This repo serves as a one-stop shop for slide decks, tutorial guides, and live demo assets used at global conventions and workshops.

## Project Overview
The Earth Observing Dashboard is a joint initiative by the European Space Agency (ESA), the National Aeronautics and Space Administration (NASA), and the Japan Aerospace Exploration Agency (JAXA).

This tri-agency effort converges Earth Observation (EO) data from complementary satellite missions to share scientifically grounded insights with the global public. By leveraging interactive storytelling, the dashboard combines:

- Interactive Maps: Visualizing global changes in real-time.

- Scientific Indicators: Tracking environmental shifts.

- Dynamic Charts: Exploring the impacts of global events on our planet.

## Repository Structure
To keep things organized, materials are categorized by event type:

- `/presentations:` Slide decks (PDF) or scripted walkthroughs and speaker notes for major conferences.

- `/tutorials:` Step-by-step guides for users and developers to navigate the dashboard.

## Getting Started
Check for Updates: We frequently update these materials ahead of major Earth Science conventions (e.g., AGU, EGU, IGARSS).
